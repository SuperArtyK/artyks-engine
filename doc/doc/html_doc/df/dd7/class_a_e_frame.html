Detailed Description

This module is used for creating delays(in FPS format) in the engine.

It is great if used in loops, as it makes them have delay of AEFrame, if body executes fasted than the delay time. This module is not inherited from the base class, as it needs to be very fast, faster than the base class even, and without much of dependencies.
